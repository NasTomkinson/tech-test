# Description
This project forms part of a tech test for an application. The goal was to make the front-end for a web banking application with mock backend data.

## Project stack
This project uses NextJS, Tailwind, TailwindCSS and Storybook. 

## Running the application
Installation instructions assume you already have node and git installed on your machine. <br />
This project uses node version `22.22.2`

- Clone this repoistory using `git clone https://github.com/NasTomkinson/tech-test.git .`
- Run `npm install` to install all npm packages

<b> For development </b>
- Run `npm run dev && npm run storybook` to start the development server and storybook application.

<b> For production </b>
- Run `npm run build && npm run start` to run the production ready version of this application. 
 
<b> Testing </b>
- Run `npm run test` to run test suites within the application. This will also provide you with a test coverage table.
